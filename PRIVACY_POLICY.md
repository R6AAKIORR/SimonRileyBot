# Política de Privacidade – SimonRileyBot

Última atualização: 13 de junho de 2026

Esta Política de Privacidade descreve como o SimonRileyBot lida com informações ao ser utilizado em servidores do Discord.

## 1. Coleta de Dados

O SimonRileyBot não coleta informações pessoais sensíveis dos usuários.

Para o funcionamento de determinados comandos, o bot pode acessar informações disponibilizadas pela API do Discord, como:

* Nome de usuário;
* ID de usuário;
* Nome de servidor;
* Avatar do usuário;
* Informações públicas fornecidas pelo Discord.

## 2. Uso das Informações

As informações acessadas são utilizadas exclusivamente para fornecer as funcionalidades do bot, responder a comandos e exibir informações solicitadas pelos usuários.

## 3. Armazenamento de Dados

Atualmente, o SimonRileyBot não armazena permanentemente dados pessoais dos usuários, salvo quando necessário para o funcionamento de recursos específicos.

## 4. Compartilhamento de Dados

O SimonRileyBot não vende, aluga ou compartilha dados de usuários com terceiros.

## 5. Segurança

São adotadas medidas razoáveis para proteger as informações utilizadas pelo bot. Entretanto, nenhum sistema é completamente livre de riscos.

## 6. Alterações nesta Política

Esta Política de Privacidade pode ser atualizada a qualquer momento. O uso contínuo do bot após alterações constitui aceitação da versão mais recente.

## 7. Contato

Em caso de dúvidas sobre esta Política de Privacidade, entre em contato com o desenvolvedor do SimonRileyBot através do Discord.
